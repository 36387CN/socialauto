# 0广告费获客计划 — 90天10,000客户

## 核心逻辑

不花1分钱广告费，用产品本身 + 内容 + 社群驱动增长。

增长引擎:
  内容SEO → 百度排名获取自然流量
  推荐裂变 → 每个用户带来N个新用户
  免费模板 → 吸引精准用户注册
  KOL互换 → 零成本互相导流
  社群运营 → 微信群沉淀 + 裂变

## 增长引擎2：推荐裂变 (获客占比30%)

### 裂变机制
- 每个注册用户获取唯一推荐链接 `/ref/{code}`
- 推荐好友注册 → 推荐人和好友各得 30天免费使用
- 推荐5人 → 150天免费 (近半年)
- 推荐10人 → 300天免费 (近一年)

### 裂变放大方法
1. 用户分享推荐链接到微信群的弹出提示 (产品内引导)
2. "推荐排行榜"公开显示 (社交压力驱动)
3. 推荐满3人解锁高级功能 (AI文案高级版)
4. 用户发布的内容自动带"由社媒自动化助手生成"小尾巴 (品牌曝光+被动推荐)

## 增长引擎4：社群运营 (获客占比10%)

### 微信群矩阵
1. 建"小企业社媒运营交流群"，每天在群内分享干货
2. 群内发起话题讨论，提高活跃度
3. 每周五线上分享会 (15分钟语音分享)
4. 群成员邀请朋友进群 → 群裂变
5. 群公告放产品注册链接

### 冷启动方法
- 前50个种子用户：手动邀请身边开小店/做微商的朋友
- 在"小企业主""创业者"等QQ群/微信群分享内容

- 成本：0元，双方都获益
- 目标：每月合作5-10个账号

## 零成本获客的底层逻辑

不用钱买流量，用时间换流量。
每天3-4小时内容输出 + 社群运营 + 产品迭代。
前2周最苦，几乎没有用户。
第3周开始势能积累，第6周进入自增长轨道。
关键是：**每天不断，雷打不动。**
